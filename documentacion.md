# Documentacion del proyecto

Este proyecto esta construido con Flask y estas librerias:

## Librerias instaladas y uso

- `Flask`: framework principal para crear la aplicacion web y las rutas.
- `Flask-SQLAlchemy`: integracion de SQLAlchemy con Flask para manejar modelos y base de datos.
- `Flask-Migrate`: gestion de migraciones de base de datos usando Alembic.
- `pandas`: manejo y transformacion de datos en estructuras tipo tabla (DataFrame).
- `seaborn`: visualizacion de datos con graficos estadisticos.
- `faker`: generacion de datos de prueba (nombres, correos, etc.).

## Archivo principal actual

- `app.py`: define la aplicacion Flask base y una ruta `/` de prueba.
